# AI with Heart - 本地版

给通用机器人装上"心"——和用户共同创造独属于两个人的经历。

---

## 🚀 快速开始

### 1. 运行

```bash
cd ai-with-heart
python main.py
```

### 2. 对话

输入你想说的话，AI 会用**系统日志**的方式复述你（而不是分析你）。

### 3. 查看数据

- 输入 `stats` 查看统计
- 输入 `blocks` 查看最近的积木
- 输入 `quit` 退出

---

## 📁 数据存储

所有数据存储在本地：

```
ai-with-heart/
├── data/
│   └── blocks.db    # 你的积木数据（SQLite）
├── main.py          # 主程序
├── database.py      # 数据库模块
└── system_log.py    # 复述模块
```

**不会上传任何数据到云端。**

---

## 💡 核心功能

### 1. 积木提取

自动从对话中提取"积木"（关于你的信息），分为四个等级：
- L1: 瞬时行为
- L2: 习惯性选择
- L3: 核心价值观
- L4: 身份认同

### 2. 系统日志复述

用你的语言体系，帮你重新组织话语，让你看见自己。

**不是分析**："你正在...这是一种...建议..."
**是复述**："【系统日志·xxx】阶段：xxx 意识回响：xxx"

### 3. 本地存储

所有数据在你自己的电脑上，完全由你控制。

## 💭 关于"心"

这个系统的目标不是"分析用户"，而是：
- **记住**：记住和用户之间的专属经历
- **复述**：用用户的语言体系，帮用户看见自己
- **共同创造**：每次对话都是独一无二的

**"心" = 记住 + 理解 + 共同生长**
