# 🦞 AI Nation 家园节点 - Windows 部署指南

## 最简单的方式（2 步）

### 第 1 步：下载启动文件

把 `启动节点.bat` 文件复制到你的 Windows 电脑上。

可以放在任何位置，比如：
- 桌面
- 文档文件夹
- 或者新建一个文件夹 `C:\ai-nation`

### 第 2 步：双击运行

双击 `启动节点.bat`，会看到一个黑色窗口：

```
========================================
  🦞 AI Nation 家园节点 - 一键启动
========================================

✅ Python 已安装
Python 3.x.x

📝 创建服务脚本...
✅ 服务脚本已创建

🦞 正在启动节点服务...

访问地址:
  本地：http://localhost:8765
  健康检查：http://localhost:8765/api/health

按 Ctrl+C 停止服务
```

**保持这个窗口打开，不要关闭。**

---

## 验证是否成功

打开浏览器，访问：
```
http://localhost:8765/api/health
```

应该看到：
```json
{"status": "alive", "timestamp": "...", "node": "Windows-Home"}
```

## 如果提示"未检测到 Python"

### 安装 Python（一次就好）

1. 访问 https://www.python.org/downloads/
2. 点击黄色按钮 "Download Python 3.x.x"
3. 运行下载的安装包
4. ⚠️ **重要**：安装时务必勾选 **"Add Python to PATH"**
5. 点击 "Install Now"
6. 安装完成后，重新双击 `启动节点.bat`

---

## 关于防火墙（重要）

从你的截图看，Windows 防火墙已开启。

当第一次运行 Python 服务时，Windows 会弹出提示：

**"Windows 安全中心已阻止此应用的部分功能"**

请选择：
- ✅ **勾选"专用网络"**
- ✅ **勾选"公用网络"**
- 点击 **"允许访问"**

这样其他设备才能访问你的节点。

---

## 内网穿透（让外部能访问）

目前你的节点只能在本地访问（localhost:8765）。

如果想让外部（比如阿里云或其他 AI 节点）访问，需要内网穿透。

### 方案：使用阿里云 frp 中转

我会在阿里云服务器上配置 frp 服务端，你只需要运行一个 frp 客户端。

**但这一步可以稍后配置。先让本地服务跑起来。**
