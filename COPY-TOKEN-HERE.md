# 📋 请复制新生成的 Token

**步骤**:

1. 点击页面上的 **Generate new token** 按钮
2. 填写：
   - Note: `ai-nation`
   - scopes: 勾选 `repo`
3. 点击 **Generate token**
4. **立即点击复制图标** 📋（在 Token 旁边）
5. 回来运行：

```bash
export GITHUB_TOKEN="粘贴刚才复制的 Token"
./scripts/setup-zero-friction.sh
```

---

**🦞 这次复制后，Token 会自动保存到 ~/.bashrc**
